# 🚗 Headcar — demo checklist (prop next to the laptop)

## ✅ BEFORE you're on stage
- [ ] Car on a **full 18650 pack** · charger / spare pack within reach
- [ ] **Phone hotspot ON** (`Jon iPhone 16`) — **NOT** the venue WiFi
- [ ] Laptop on the hotspot · dashboard open in **Chrome**
- [ ] Connect car → `ws://172.20.10.2:81`  *(raw IP — `car.local` fails on Windows)*
- [ ] Connect Muse → **Recalibrate** → **4 GREEN dots** (sit still 5 s)
- [ ] **Backup queued**: recorded run / load `recordings/headcar-2026-06-19_08-41-45.json` in the replay
- [ ] Leave it **DISARMED** until showtime

## 🎬 THE DRIVE (on stage — let each beat land)
1. **Calibrate** — 5 s, sit still → 4 green dots
2. **Arm** ("▶ Start listening") — "until now it ignored me — safety layer one"
3. **Forward** — clench **light → hard** (show the speed ramp)
4. **Reverse** — eyebrow raise
5. **Steer** — head tilt L / R
6. **Stop** — disarm + "even if my laptop died, a half-second watchdog stops it"

## 🧯 IF IT BREAKS — the 45-second rule
- **Do NOT debug live.** After ~45 s, drop it.
- Say: *"Here's the same thing from a recorded run"* → play the backup → keep moving.
- A flaky contact dot ≠ dead air. Narrate and roll.

## 👀 GLANCE RULES
- **Hands off** the keyboard while you talk — look at the **room**, not the screen.
- **Pause** after each gesture so people register it ("wait… his *face*").
- Battery: ~11 min continuous — fine for the talk; **recharge/swap during the open demo hour**.
